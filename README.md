Amazon Fresh Analytics
Project Overview
Amazon Fresh Analytics is a SQL-based database management and business analytics project developed to manage customer information, product inventory, suppliers, orders, and customer reviews.
The project demonstrates database design, SQL query development, normalization, constraints, transactions, and business insight generation.

    • Constraints
    • Joins
    • Aggregations

    • Normalization (3NF)
    • ACID Transactions
    • Business Analytics
